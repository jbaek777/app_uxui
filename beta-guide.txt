━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📲 1단계. 앱 설치 방법 (안드로이드)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① APK 파일 다운로드 (아래 링크)
   → https://expo.dev/accounts/jbaek777/projects/meatmanager/builds/72724675-4b81-40c2-a367-44255156f8c9

② 설치 전 설정 필요:
   설정 → 보안 → '알 수 없는 앱 설치' 허용
   (또는 파일 다운로드 후 "허용" 버튼 클릭)

③ 다운로드된 APK 파일 클릭 → 설치

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📝 2단계. 첫 실행 (온보딩)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

① "사장님으로 시작" 선택

② 사업장 정보 입력
   - 상호명: 매장 이름 입력
   - 대표자명: 사장님 성함
   - 사업자등록번호: 자동 검증됩니다
   - 사업자 유형: 개인사업자 / 법인사업자 선택
   - 매장 주소: 시/도 → 구/군 선택 후 동 입력 (선택)

③ 취급 축종 선택 (한우, 육우, 한돈 등 중복 선택 가능)

④ 사장님 계정 등록
   - 이름 + PIN 4자리 설정

⑤ 완료! 직원 초대 코드가 자동 생성됩니다
   (직원에게 코드 공유 → 직원은 "직원으로 참여" 선택)

━━━━━━━━━━━━━━━━━━━━━━━━━━━━
📱 3단계. 주요 기능 안내
━━━━━━━━━━━━━━━━━━━━━━━━━━━━

🏠 홈 (대시보드)
   - 오늘 위생점검 현황
   - 소비기한 임박 경고
   - 빠른 실행 버튼 6개

🏷️ 조회
   - 이력번호 바코드 스캔
   - 숙성관리 (숙성일 추적, PDF 출력)

📦 재고
   - 재고 현황 (2x2 요약 카드)
   - 판매 완료 처리 → 판매내역 탭에 자동 기록
   - 수율 계산기 (계산 이력 저장)
   - 소비기한 관리 및 수정 (수정 로그 자동 기록)

🖨️ 서류·출력
   - 위생점검 일지 작성 및 PDF 출력
   - 온도관리 기록부
   - 마감 정산서
   - 직원 보건증 현황

⚙️ 설정
   - 다크/라이트 모드 전환
   - 직원 추가 및 보건증 만료일 관리
   - 알림 설정

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

- 입력한 모든 데이터는 자동 저장됩니다
- 앱을 껐다 켜도 데이터 유지됩니다
- 인터넷 연결 시 클라우드 백업 자동 진행
