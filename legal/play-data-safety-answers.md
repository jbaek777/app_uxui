**답: 예**

### Q. 모든 사용자 데이터는 전송 중 암호화되나요?

**답: 아니요** (소규모 앱, 추후 Phase B 이상 고려)

## ❺ Play Console 입력 순서 가이드

1. **Google Play Console** 접속 → 앱 선택
2. 왼쪽 메뉴 **정책 > 앱 콘텐츠**
3. **데이터 보안** 섹션 "시작하기" 클릭
4. 순서대로 응답:
   - Step 1: 데이터 수집/공유 여부 → **예**
   - Step 2: 보안 관행 → 모두 **예**
   - Step 3: 데이터 유형 추가 → 위 ❷번 섹션의 14개 항목 순서대로 입력
   - Step 4: 검토 및 제출
5. 제출 후 대략 **2~24시간 내** 반영됨

## ❻ 자주 반려되는 사례 & 대비책

| 반려 사례 | 원인 | 대비책 |
|---|---|---|
| "수집하는데 수집 안 한다고 답변" | 실제 네트워크 요청과 불일치 | 위 템플릿 그대로 입력 (실제 코드 기반) |
| "제3자 공유 답변 누락" | Anthropic OCR 공유를 "공유 안 함"으로 답변 | 사진 항목에 "공유함" 명시 |
| "삭제 방법 없음" | 앱 내 탈퇴 기능 누락 | `SettingsScreen`에 탈퇴 버튼 필수 |
| "HTTPS 아닌 통신" | 일부 API가 HTTP | 모든 API HTTPS 확인 |
